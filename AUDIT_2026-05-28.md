## Strengths

- **Clean adapter/core/runner layering** — enforced by `test/architecture/core-boundary.test.ts`.
- **Per-database execution lock** — `runWithAccessExecutionLock` (`access-runner.ts:319-341`) serializes runs against the same accdb case-insensitively.
- **Atomic, multi-process-safe registry** — `FileAccessOperationRegistry` uses `mkdir`-as-lock + tmp-file + rename for atomic writes.
- **Secret scrubbing** — `sanitizeSecrets` applied to stdout/stderr. Passwords via env vars, never CLI args.
- **Path traversal defense** — `Resolve-SandboxedPath` (`dysflow-access-runner.ps1:265-277`) blocks `C:\root\..\evil`.
- **Read-only safety** — all read paths use `OpenRecordset()` which cannot execute DDL/DML by DAO design. SQL guard in tools.ts is defense-in-depth. ✅ **Now also has TypeScript-layer keyword guard.**
- **AutoExec neutralization** — `Disable-StartupFeatures` with CRITICAL abort.
- **Test suite breadth** — 49 files, 623 tests.

## Technical Debt

### 1. PowerShell God Script — HIGH ⬜ OPEN

`dysflow-access-runner.ps1` is 1700 lines with **two parallel read-dispatch blocks**:
- Early dispatch: lines 1292-1383 (6 read actions)
- Main block: lines 1385-1700 (12+ read actions)

Adding a read action requires updating both blocks. Drift already visible.

### 2. Legacy Tool Dispatch Cascade — HIGH ⬜ OPEN

4 separate `LEGACY_*_TOOL_NAMES` arrays + `createLegacyDispatchTool` cascade + parity registry = **5 places to keep in sync**. Duplicate between `tools.ts:722-740` and `legacy-tool-inventory.ts:25-50`.

**Fix**: `Record<name, {mode, requestBuilder, schema}>` — ~150 lines removed.

### 3. Two Registry Implementations — MEDIUM ⬜ OPEN

`InMemoryAccessOperationRegistry` and `FileAccessOperationRegistry` share no abstract base. Eviction and deep-clone duplicated (`access-operation-registry.ts:279-286` vs `334-340`).

### 4. `[System.GC]::Collect()` After Every DAO Close — MEDIUM ⬜ OPEN

Workaround for COM RCW leaks. 100-500ms overhead per call. Scales poorly on `relink_directory`.

### 5. `_handleValidatedLegacyQuery` Dead Code — LOW ⬜ OPEN

Exported with `_` prefix (`tools.ts:115-123`). Unused in a hot module.
